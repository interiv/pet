## 💡 问题分析

### 问题1: 知识点统计API失败

**可能原因**:
1. 路由未正确注册到server.js
2. API路径与前端调用不一致
3. 数据库表不存在

**验证方法**:
```bash
# 检查路由注册
grep -n "knowledge-points" backend/src/server.js

# 检查数据库表
node -e "const db=require('better-sqlite3')('./data/database.sqlite'); console.log(db.prepare('SELECT name FROM sqlite_master WHERE type=\"table\"').all())"
```

**解决方案**:
确认server.js中有以下代码:
```javascript
app.use('/api/knowledge-points', knowledgePointRoutes);
```

---

### 问题2: 技能API失败

**可能原因**:
1. 路由未注册
2. 技能表未初始化

**验证方法**:
```bash
# 检查路由注册
grep -n "skills" backend/src/server.js

# 检查技能数据
node -e "const db=require('better-sqlite3')('./data/database.sqlite'); console.log(db.prepare('SELECT COUNT(*) FROM skills').get())"
```

---

### 问题3: 创建作业失败

**错误信息**: "请填写必要信息"

**可能原因**:
- 请求参数不完整
- API需要额外的字段

**解决方案**:
检查assignments.js路由,确认必需参数:
```javascript
// 可能需要:
{
  class_id: number,
  subject: string,
  title: string,
  // 可能还需要其他字段
}
```
