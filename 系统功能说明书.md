# 班级宠物养成系统 - 功能说明书

## 一、系统概述

班级宠物养成系统是一个将学习与游戏化结合的在线教育平台，通过完成作业获得经验值来培养专属宠物，支持宠物对战、社交互动、成就系统等丰富的游戏化元素。

### 技术架构
- **后端**：Node.js + Express + SQLite
- **前端**：React + TypeScript + Ant Design
- **认证**：JWT Token
- **实时通信**：Socket.IO（可选）

---

## 二、角色说明

系统包含四种角色，每种角色拥有不同的功能权限：

| 角色 | 说明 | 审批要求 |
|------|------|----------|
| 学生 | 普通用户，主要使用者 | 注册后需班主任审批 |
| 任课教师 | 教授课程的教师 | 注册后需管理员审批 |
| 班主任 | 班级管理者 | 管理员指定 |
| 管理员 | 系统管理者 | 初始创建 |

## 三、功能详解

### 3.1 学生功能

#### 3.1.1 用户模块
- **注册登录**：用户名密码注册，选择要加入的班级，等待审批后激活
- **个人中心**：查看/修改个人信息、头像

#### 3.1.2 宠物系统
| 功能 | 说明 |
|------|------|
| 创建宠物 | 选择一种宠物（30+种），每种有独特进化形态 |
| 宠物属性 | 等级、经验、攻击力、防御力、速度、体力、饱腹度、心情、健康值 |
| 成长阶段 | 幼年期(1-10级) → 青年期(11-30级) → 成年期(31-50级) → 完全体(51-80级) → 究极体(81-100级) |
| 喂食 | 使用背包道具喂食，恢复饱食度和心情 |
| 装备 | 为宠物穿戴武器、防具、饰品 |

#### 3.1.3 作业系统
| 功能 | 说明 |
|------|------|
| 查看作业 | 查看班级作业列表、截止时间 |
| 做作业 | 选择题目作答，支持单选/多选/判断/简答 |
| 图片上传 | 简答题可上传作答图片 |
| 提交批改 | AI自动批改，获得经验奖励和金币奖励 |
| 错题本 | 记录做错的题目，可重新作答 |
| 重做错题 | 针对错题进行专项练习 |

#### 3.1.4 战斗系统
| 功能 | 说明 |
|------|------|
| 寻找对手 | 查看全校/全班可挑战的宠物 |
| 发起挑战 | 1v1回合制对战，根据属性和装备计算胜负 |
| 战斗奖励 | 胜利获得经验、金币奖励 |
| 战斗记录 | 查看历史对战记录 |

#### 3.1.5 社交系统
| 功能 | 说明 |
|------|------|
| 添加好友 | 通过用户名搜索并添加好友 |
| 好友列表 | 查看好友基本信息、亲密度 |
| 访问宠物 | 查看好友的宠物详情和装备 |
| 赠送礼物 | 送给好友道具增加亲密度 |
| 好友对战 | 与好友进行宠物对战 |

#### 3.1.6 商店系统
| 功能 | 说明 |
|------|------|
| 道具商店 | 购买各种道具（体力药剂、治疗药剂、经验药水等） |
| 我的背包 | 查看拥有的道具，数量及效果 |
| 使用道具 | 投喂宠物或使用道具 |

#### 3.1.7 成就系统
| 分类 | 说明 |
|------|------|
| 战斗成就 | 胜利次数、连胜次数等 |
| 宠物成就 | 等级提升、进化阶段 |
| 社交成就 | 好友数量、亲密度 |
| 收集成就 | 收集物品、装备 |
| 特殊成就 | 特殊条件触发 |

#### 3.1.8 排行榜
- **等级排行榜**：按宠物等级排序
- **班级排行榜**：按班级总经验排序（管理员/教师可见）

---

### 3.2 任课教师功能

#### 3.2.1 工作台总览
- 查看所负责班级的统计数据
- 班级数量、学生数量一览

#### 3.2.2 学生管理
- 查看所教班级的学生列表
- 查看学生详情（宠物、背包、装备）
- 禁用/启用学生账号

#### 3.2.3 班级管理
- 查看所教班级列表
- 查看班级学生人数、班级总经验
- 为班级添加其他任课教师

#### 3.2.4 入学申请管理
- 查看所教班级的入学申请
- 审批（批准/拒绝）学生入学申请
- 筛选待审批/已批准/已拒绝的申请

#### 3.2.5 数据查看
- 查看班级宠物排行榜
- 查看班级学生作业情况

#### 3.2.6 作业管理（需管理员开启权限）
- 创建作业（AI生成题目）
- 布置作业给班级
- 查看作业统计

---

### 3.3 班主任功能

班主任是任课教师的特殊角色，拥有以下额外权限：

#### 3.3.1 入学申请审批
- **独有权限**：可以审批新学生的入学申请
- 审批学生加入班级
- 审批教师加入班级（作为任课教师）

> 注意：普通任课教师无法审批入学申请，只有班主任可以。

## 五、核心业务流程

### 5.1 用户注册流程

```
1. 用户注册 → 选择角色（学生/教师）→ 选择班级
2. 系统创建申请记录，状态为"待审批"
3. 学生 → 班主任审批 → 批准后加入班级
4. 教师 → 管理员审批 → 批准后激活账号
```

### 5.2 作业完成流程

```
1. 教师创建作业 → 设置科目、难度、截止时间
2. AI生成题目（如需要）
3. 学生查看作业 → 作答（支持图片上传）
4. 提交作业 → AI批改 → 获得经验和金币
5. 错题自动录入错题本
6. 学生可重新作答错题
```

### 5.3 宠物养成流程

```
1. 达到条件（如作业得分）→ 获得经验
2. 经验达标 → 宠物升级
3. 等级达标 → 宠物进化（新形态）
4. 购买/获得装备 → 为宠物穿戴
5. 宠物属性提升 → 战斗力增强
```

### 5.4 战斗流程

```
1. 发起挑战 → 选择对手
2. 系统计算属性（攻击+防御+速度+装备加成）
3. 计算胜率 → 生成战斗结果
4. 胜利方获得经验、金币奖励
5. 更新战斗记录
```

## 六、游戏化元素

### 6.1 经验获取途径
- 完成作业：主要经验来源
- 战斗胜利：额外经验奖励
- 成就奖励：完成成就获得经验
- 每日签到：持续登录奖励

### 6.2 金币用途
- 购买道具
- 未来：购买宠物皮肤、装饰等

### 6.3 宠物进化条件
- 达到指定等级自动进化
- 进化后形象改变
- 属性获得大幅提升

### 6.4 成就系统
- 战斗成就：累计胜利次数、连胜记录
- 宠物成就：达到特定等级、进化次数
- 社交成就：好友数量、赠送礼物次数
- 收集成就：收集特定物品、装备
- 特殊成就：稀有条件触发

## 八、数据安全

- 密码加密存储（bcrypt）
- JWT Token认证
- 角色权限验证
- 学生数据隔离（只看本班数据）
- 操作日志记录

## 九、系统特点

1. **游戏化学习**：将学习过程游戏化，提高学生积极性
2. **AI智能批改**：自动批改作业，减轻教师负担
3. **班级社交**：支持好友系统，增加互动性
4. **实时反馈**：完成任务立即获得奖励反馈
5. **多角色支持**：清晰的角色权限划分
6. **数据可视化**：丰富的统计和排行榜
