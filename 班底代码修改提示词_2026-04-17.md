### 1.4 每日任务系统
**目标文件**: `backend/src/routes/daily-tasks.js`（新建）, `frontend/src/components/DailyTasks.tsx`（新建）

**后端接口**:
```javascript
// GET /api/daily-tasks - 获取今日任务
// POST /api/daily-tasks/claim - 领取任务奖励（需完成任务条件）

// 任务类型：
// - complete_assignment: 完成N道作业
// - correct_rate: 正确率达到X%
// - feed_pet: 投喂宠物N次
// - login: 每日登录（自动完成）
// - streak: 连续N天完成任务（周卡/徽章）

// 返回数据结构：
{ tasks: [...], streak_days: 3, claimable_rewards: [...] }
```

**数据库表**:
```sql
CREATE TABLE daily_tasks (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  date TEXT,        -- YYYY-MM-DD
  tasks_completed INTEGER,
  streak_days INTEGER,
  last_updated TEXT
);
```

**前端**:
- 首页侧边栏增加"每日任务"入口
- 任务面板显示今日任务进度和可领取奖励
- 需完成的任务前置检查（如"完成作业"需API验证）
