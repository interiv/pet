# 班级宠物养成系统实现计划（完善版）

## 项目概述
为 50+ 学生的班级创建一个智能化宠物养成系统，通过作业完成情况（支持 AI 自动批改）投喂升级，支持宠物互动、战斗、个人账号管理和班级展示看板。

## 技术栈选择
- **前端**: React + TypeScript + Vite
- **UI 框架**: Ant Design (适合后台管理系统)
- **后端**: Node.js + Express
- **数据库**: SQLite (轻量级，适合 50 人规模)
- **状态管理**: Zustand (轻量级状态管理)
- **样式**: Tailwind CSS
- **AI 集成**: 大模型 API（用于作业批改、智能对话）
- **实时通信**: Socket.io (战斗、互动实时性)

## 核心功能模块

### 1. 用户系统
- 学生账号注册/登录
- 教师账号（布置作业、查看统计）
- 个人信息管理
- 家长绑定（可选，查看孩子进度）

### 2. 宠物系统
- **宠物形象设计**（50+ 种不同宠物）
  - 初始宠物选择：12 星座对应 12 种基础宠物
  - 隐藏宠物：通过特殊成就解锁
  - 季节限定宠物：活动专属
- **宠物属性**：等级、经验值、饥饿度、心情值、健康值、体力值
- **宠物成长阶段**：幼年→青年→成年→完全体→究极体
- **宠物类型**：
  - 攻击型（高攻击、低防御）
  - 防御型（高防御、低速度）
  - 平衡型（各项均衡）
  - 速度型（高速度、高闪避）
  - 特殊型（特殊技能）

### 3. AI 集成系统（新增核心功能）
- **AI 作业批改**
  - 主观题自动批改（作文、简答、编程题）
  - 客观题自动批改（选择、填空、判断）
  - 批改反馈生成（指出错误、给出建议）
  - 批改置信度显示（低置信度转人工复核）
- **AI 宠物助手**
  - 智能对话：解答学习问题
  - 学习建议：根据作业情况给出建议
  - 宠物互动：宠物会说鼓励的话
- **AI 出题系统**
  - 根据学习进度生成练习题
  - 个性化题库推荐
  - 难度自适应调整

### 4. 升级系统
- **经验获取**:
  - 完成作业获得经验值（AI 批改后自动发放）
  - 日常签到奖励
  - 连续登录奖励
  - 成就奖励
- **投喂系统**:
  - **食物类**：
    - 普通粮食：+10 经验
    - 高级零食：+50 经验
    - 特殊料理：+100 经验 + 心情提升
    - 限时食物：双倍经验效果（30 分钟）
    - 学霸套餐：+200 经验（需高正确率作业）
  - **道具类**：
    - 经验药水：立即增加经验
    - 心情药水：提升心情值
    - 治疗药剂：恢复健康值
    - 体力药剂：恢复体力值
    - 保护罩：防止战斗失败掉落经验
- **装备系统**:
  - **服装**：
    - 帽子：学士帽、运动帽、王冠等
    - 衣服：校服、礼服、铠甲、法袍等
    - 鞋子：运动鞋、靴子、翅膀等
    - 饰品：项链、戒指、徽章等
  - **武器**：
    - 近战类：剑、斧、拳套
    - 远程类：弓、法杖、枪械
    - 特殊类：扇子、乐器、书籍
  - **防具**：
    - 头盔：增加防御
    - 铠甲：大幅增加防御
    - 盾牌：格挡率提升
  - **套装效果**：收集全套触发额外加成

### 5. 互动系统
- **宠物拜访**：访问同学的宠物小屋
- **互动动作**：抚摸、喂食、玩耍、合影
- **亲密度系统**：提升同学间宠物亲密度
- **宠物托管**：外出时可托管给好友照顾
- **合作任务**：好友组队完成特殊任务
- **礼物系统**：互赠道具增加亲密度

### 6. 战斗系统（回合制）
- **1v1 对战模式**：好友约战、随机匹配、天梯赛
- **3v3 团队战**：班级对抗赛
- **属性克制**：火→草→水→火，光↔暗
- **技能系统**：
  - 每个宠物 3-4 个主动技能
  - 1-2 个被动技能
  - 技能升级系统
- **战斗奖励**：
  - 获胜获得经验、金币、道具
  - 连胜额外奖励
  - 赛季排名奖励
- **战斗动画**：
  - 技能特效
  - 伤害数字
  - 宠物表情变化

### 7. 看板系统
- **个人宠物展示**：3D/2D 展示台，可旋转查看
- **班级宠物排行榜**：
  - 等级排行榜（前 10 名）
  - 战斗胜率排行榜
  - 作业完成度排行榜
  - 亲密度排行榜
  - 收藏排行榜（装备、宠物数量）
- **成就墙**：展示班级同学获得的稀有成就
- **最新动态**：宠物诞生、升级、战斗胜利等动态

### 8. 作业系统（AI 增强）
- **作业发布**（教师端）：
  - 支持多种题型：选择、填空、判断、简答、作文、编程
  - 设置截止日期和满分经验值
  - 上传附件（图片、文档）
  - AI 辅助出题
- **作业提交**（学生端）：
  - 在线作答
  - 上传文件（作文拍照、代码文件）
  - 草稿自动保存
- **作业批改**：
  - **客观题**：自动批改（即时出结果）
  - **主观题**：AI 批改（使用大模型 API）
    - 作文批改：语法、结构、内容评价
    - 编程题：功能测试 + 代码质量评价
    - 简答题：关键词匹配 + 语义理解
  - **人工复核**：低置信度标记，教师手动复核
  - **批改反馈**：详细错误分析 + 改进建议
- **错题本**：
  - 自动收集错题
  - 相似题推荐
  - 错题重做功能

### 9. 商店系统
- **金币商店**：购买基础道具、装备
- **钻石商店**（充值/成就获得）：购买稀有物品
- **限时商店**：季节性、节日限定商品
- **二手市场**：同学间交易装备道具

### 10. 成就系统
- **学习成就**：
  - 作业达人：连续 7/30/100 天完成作业
  - 学霸之路：作业正确率 90%/95%/100%
  - 进步之星：成绩提升幅度
- **战斗成就**：
  - 战斗王者：10/50/100 连胜
  - 一击必杀：单次技能造成大量伤害
  - 逆转胜利：残血反杀
- **社交成就**：
  - 社交达人：好友数量达到 10/30/50
  - 人气王：被拜访次数
  - 最佳搭档：与固定好友亲密度
- **收集成就**：
  - 收藏家：收集装备数量
  - 宠物大师：拥有所有宠物
  - 套装收集：收集完整套装

### 11. 任务系统
- **每日任务**：
  - 登录签到
  - 完成 1 次作业
  - 进行 1 次战斗
  - 拜访 3 位好友
- **每周任务**：
  - 完成 5 次作业
  - 战斗胜利 10 场
  - 提升宠物等级
- **成就任务**：
  - 长期目标，奖励丰厚
- **活动任务**：
  - 节日限定任务
  - 班级活动任务

### 12. 班级系统
- **班级信息页**：
  - 班级成员列表
  - 班级平均等级
  - 班级荣誉墙
- **班级活动**：
  - 班级 BOSS 战（合力挑战）
  - 班级竞赛（学习 + 战斗综合）
  - 班级建设（贡献度系统）
- **公告系统**：
  - 教师发布公告
  - 活动通知
  - 作业提醒

## 宠物形象设计策略

### 设计风格
- **像素风格**：复古、易实现、资源占用小
- **Q 版卡通**：可爱、亲和力强
- **低多边形**：现代感、3D 效果好

### 多样性保证
1. **基础模板 + 变体**：
   - 12 种基础宠物（对应 12 星座）
   - 每种 3-5 种颜色变体
   - 不同性别差异

2. **成长阶段差异**：
   - 幼年：体型小、细节少
   - 青年：体型中等、特征明显
   - 成年：完全形态、细节丰富
   - 完全体：特效环绕、气势强
   - 究极体：华丽变身、专属特效

3. **装备外观**：
   - 不同装备实时改变外观
   - 武器可视化
   - 服装搭配系统

4. **特殊状态**：
   - 战斗状态：眼神变犀利
   - 开心状态：表情愉悦
   - 疲惫状态：打哈欠
   - 生病状态：脸色差

### 实现方案
- 使用 Aseprite 制作像素动画
- Blender 制作 3D 模型（可选）
- AI 生成基础形象后人工精修
- 购买部分素材包降低成本

## 升级元素详细设计

### 等级系统 (1-100 级)
```
等级区间：
- 1-10 级：新手期（快速升级）
- 11-30 级：成长期（正常速度）
- 31-50 级：成熟期（难度提升）
- 51-80 级：精英期（难度较高）
- 81-100 级：大师期（极难）

每级所需经验公式：exp = base_exp * (level ^ 1.5)
每 10 级：突破任务 + 外观变化
特殊等级（50/80/100）：专属奖励
```

### 属性成长
```
每次升级获得属性点：5 点
可自由分配：攻击、防御、速度、体力
不同宠物类型有属性上限
重置属性道具：允许重新分配
```

### 技能系统
```
技能学习：
- 等级解锁：达到指定等级自动学习
- 技能书：使用技能书学习
- 任务奖励：完成特殊任务

技能升级：
- 使用技能点升级
- 重复使用技能增加熟练度
- 技能融合：两个同级技能合成高级技能
```

### 亲密度系统
```
亲密度等级：
- 陌生 (0-100)
- 认识 (101-300)
- 朋友 (301-600)
- 好友 (601-1000)
- 挚友 (1001-2000)
- 羁绊 (2001+)

提升方式：
- 互动：抚摸、玩耍
- 送礼：赠送喜欢的物品
- 合作：一起完成任务
- 战斗：并肩作战

亲密度效果：
- 战斗属性加成
- 解锁特殊互动
- 解锁合作技能
```

## 技术难点与解决方案

### 1. 并发处理
- 50 人并发压力不大，但需考虑峰值
- 使用 Redis 缓存热点数据
- 数据库连接池优化
- 关键操作事务处理

### 2. AI 集成
- API 调用频率限制：使用队列 + 限流
- 成本控制：批量处理 + 结果缓存
- 响应时间：异步处理 + 进度提示
- 数据隐私：脱敏处理 + 加密传输

### 3. 战斗同步
- 回合制降低实时性要求
- WebSocket 实现状态同步
- 战斗录像回放功能
- 断线重连机制

### 4. 数据安全
- 密码 bcrypt 加密
- JWT Token 鉴权
- 防刷分机制：行为分析 + 异常检测
- 数据备份：定时备份 + 增量备份

### 5. 性能优化
- 图片懒加载
- 分页加载列表
- CDN 加速静态资源
- 数据库索引优化

### 6. 扩展性
- 微服务架构预留
- 插件化功能模块
- 配置化数值系统
- 多班级支持
