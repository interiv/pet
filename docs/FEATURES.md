## 🎮 游戏数值设计

### 宠物基础数值
```
初始属性（1 级）：
- 攻击力：10-15（根据类型）
- 防御力：8-12
- 速度：10-15
- 健康值：100
- 心情值：100
- 饥饿度：100
```

### 升级数值
```
每级属性点：5 点
升级经验公式：exp = 100 * (level ^ 1.5)
```

### 物品效果
```
普通粮食：+10 EXP
高级零食：+50 EXP
特殊料理：+100 EXP + 心情
经验药水：+200 EXP
```

## 📝 使用说明

### 基础使用流程
1. 访问 http://localhost:5173
2. 注册账号
3. 选择初始宠物
4. 完成新手引导
5. 开始学习（完成作业）
6. 培养宠物
7. 参与对战
8. 查看排行榜

### 教师使用流程
1. 注册教师账号
2. 创建班级
3. 发布作业
4. 查看学生进度
5. AI 批改作业
6. 发放奖励
