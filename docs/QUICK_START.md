# 🎉 班级宠物养成系统 - 启动指南

## 🚀 快速启动（推荐）

### 方式一：双击启动脚本
1. 双击项目根目录下的 `start.bat` 文件
2. 等待自动安装依赖和启动服务
3. 打开浏览器访问 http://localhost:5173

### 方式二：手动启动

#### 步骤 1：安装 Node.js
如果未安装 Node.js，请访问 https://nodejs.org/ 下载并安装 **LTS 版本**（16.x 或更高）。

验证安装：
```bash
node -v
npm -v
```

#### 步骤 2：启动后端
```bash
# 打开命令行，进入后端目录
cd c:\Users\Administrator\Desktop\pet\backend

# 安装依赖（首次运行）
npm install

# 初始化数据库（首次运行）
node src/config/seed.js

# 启动开发服务器
npm run dev
```

看到以下信息表示成功：
```
数据库初始化成功
服务器运行在端口 3000
```

#### 步骤 3：启动前端
**打开新的命令行窗口**：
```bash
# 进入前端目录
cd c:\Users\Administrator\Desktop\pet\frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

看到以下信息表示成功：
```
VITE ready in xxx ms
➜  Local:   http://localhost:5173/
```

#### 步骤 4：访问系统
在浏览器中打开：**http://localhost:5173**

## 📝 使用流程

### 1. 注册账号
- 点击"立即注册"
- 填写用户名（至少 3 个字符）
- 填写密码（至少 6 个字符）
- 选择角色（学生/教师）
- 点击"注册"

### 2. 创建宠物
注册成功后会自动进入宠物创建页面：
- 给宠物起个名字（如：小火、阿水）
- 选择宠物种类：
  - 🦁 火焰狮（火属性，高攻击）
  - 🐢 水灵龟（水属性，高防御）
  - 🦌 森林鹿（草属性，高速度）
  - 🦅 光明鸟（光属性，平衡型）
  - 🐺 暗影狼（暗属性，高暴击）
- 点击"开始养成之旅"

### 3. 开始使用

#### 学生用户
- 📚 **完成作业**：获得经验值的主要途径
- 🍖 **投喂宠物**：使用物品增加经验或属性
- ⚔️ **进行对战**：与其他同学的宠物战斗
- 👫 **添加好友**：输入好友用户名添加
- 🏆 **查看排行**：了解自己的排名

#### 教师用户
- 📝 **发布作业**：创建新的作业任务
- 📊 **查看提交**：查看学生作业完成情况
- ✅ **批改作业**：AI 批改或手动批改
- 📈 **班级统计**：查看班级整体进度

## 🛠️ 常见问题

### Q1: 端口被占用怎么办？
**解决方法**：
1. 关闭占用端口的程序
2. 或修改配置：
   - 后端：编辑 `backend\.env`，修改 `PORT=3000` 为其他端口
   - 前端：编辑 `frontend\vite.config.js`，修改 `port: 5173`

### Q2: npm install 失败？
**解决方法**：
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
npm install
```

### Q3: 数据库初始化失败？
**解决方法**：
```bash
# 确保创建了 data 目录
mkdir data

# 重新运行初始化
node src/config/seed.js
```

### Q4: 前端页面空白？
**解决方法**：
1. 检查后端是否启动
2. 打开浏览器开发者工具（F12）查看错误
3. 清除浏览器缓存

### Q5: 如何重置所有数据？
**解决方法**：
```bash
# 删除数据库文件
cd backend
del data\pets.db

# 重新初始化
node src/config/seed.js
```
